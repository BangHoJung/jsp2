외부,내부 모듈 연계 방법(EAI) - 종류 선택

인터페이스 데이터 표준 확인
 - 테이블 정의서를 확인하고 정의서대로 json으로 데이터를 변환한 형태로 작성
    ex) {"eno" : "TQ98", "name" : "강병현" , .... }


보안 취약점에 대한 시큐어 코딩 항목

-------------  작업형 문제 -------------------
jsp에서 폼에 데이터를 입력 받아서 ajax로 데이터 전송
 - ex) 학번은 8자리, 문자열  --- 데이터 타입 알려줌
 - 받은 데이터를 테이블에 추가
   -- 페이지 이름, 서블릿 클래스, dao 클래스는 이름을 미리 지정해놈
 - 데이터 등록시에 데이터가 테이블에 저장되기에 문제가 없거나 조건에 맞을때만 저장됨.
 - 조건에 안맞거나 문제있는 데이터는 Exception처리(ajax에서 최종 처리)
   -- exception 발생시 에러코드만 ajax로 전송하여 에러코드에 따라 경고메시지 띄움

해당 테이블에 있는 내용을 입력폼 아래에다가 전부 출력
 - 출력할 데이터는 ajax를 이용해서 전달

JUnit 테스트 진행시 주어진 테스트용 DAO 클래스 데이터 파일을 읽어서 등록한 후에 테스트 수행.
테스트 종료후에 해당 데이터를 삭제
 - 테스트할 메소드는 특정메소드로 지칭함.
